Summary of Decisions

Header structure
Logo (36px) | Title + Subtitle | Right actions — same skeleton everywhere
Sticky?
Yes, always. Thin 1px border-bottom on non-workout pages. Progress bar replaces it on workout.
Back navigation
Recommendation: Breadcrumb subtitle (A) — "‹ Mesocycles" in the subtitle. Discoverable, no extra UI, keeps header clean.
Workout "mode"
Progress bar + RIR badge + expandable meso grid drawer. Set counter in info strip. These elements don't exist on other pages = automatic mode differentiation.
Logo behavior
Always navigates home (consistent). Never changes appearance or role.
Right-side actions
Contextual: "+ Add" on lists, "Active" badge on meso detail, RIR badge on workout, nothing on dashboard.
