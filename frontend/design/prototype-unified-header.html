Unified Header — Exploration

Three approaches to a consistent header across Dashboard, List pages, Detail pages, and active Workout. Each row shows the same variation in all four page contexts.

A — Logo-Anchored
Logo always present on the left — acts as home button on sub-pages. Back chevron appears to the left of the logo on detail/workout pages. All headers are sticky with a subtle bottom border. Workout mode appends RIR + progress bar as an extension of the same header block.

Design Tradeoffs

A — Logo-Anchored
+ Strong brand presence
+ Logo as home shortcut
+ Most visual consistency
- Header takes more vertical space
- Back + logo can feel crowded
- Logo on every page may feel heavy
B — Minimal Bar
+ Smallest footprint
+ Centered title = native feel
+ Clean swap: logo ↔ back
- Centered title truncates long names
- Tiny logo loses detail
- Workout expansion feels bigger jump
C — Contextual Strip
+ Max content space on sub-pages
+ Workout feels like "entering a mode"
+ Closest to current implementation
- Less unified — dashboard ≠ sub-pages
- No persistent brand element
- List/detail headers very bare
